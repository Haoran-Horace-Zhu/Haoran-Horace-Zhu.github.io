# Haoran Zhu — Personal Academic Website

Source code for [haoran-horace-zhu.github.io](https://haoran-horace-zhu.github.io), the academic website of Haoran Zhu, a PhD student in Mathematics at Nanyang Technological University.

The site highlights research in combinatorics, probability, and mathematical physics, together with publications, academic activities, and a short CV.

For future edits, see [UPDATE_GUIDE.md](UPDATE_GUIDE.md).

Built with [Jekyll](https://jekyllrb.com/) and the [al-folio](https://github.com/alshedivat/al-folio) theme.
